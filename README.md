# Books-that-I-like
看闲书已经有些年头，方法五花八门。
如今的大数据时代下，有些书自然还是PDF存档一下方便。
方便与人推荐，也方便自己随时能看到。
分类全凭喜好。
近期先归纳以前看过的好书pdf归档。
以后的慢慢更新。
大家如果有好书也欢迎pull。
